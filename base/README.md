# MagiskFakesip

> [FakeSIP](https://github.com/MikeWang000000/FakeSIP) is a tool used to disguise your UDP traffic as SIP protocol to evade DPI detection, using Netfilter Queue (NFQUEUE).

> [MagiskFakesip](https://github.com/Droid-MAX/magisk-fakesip) lets you run fakesip on boot with [Magisk](https://github.com/topjohnwu/Magisk)

## Supported architectures
- `arm32v7`, `arm64`, `i686`, `x86_64`

## How fast are fakesip updates?
Instant! This module is hooked to the official FakeSIP build process

## For issues and more information, check out the [main project repo](https://github.com/Droid-MAX/magisk-fakesip)
